HTML & CSS Cheat Sheet

HTML: Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
</head>
<body>
  
</body>
</html>

HTML: Common Elements

<h1>–<h6>
Headings, h1 is largest.
<p>
Paragraph.
<div>
Block container.
<span>
Inline container.
<blockquote>
Block quotation.
<hr>
Thematic break.

HTML: Text Formatting

<strong>
Strong importance (bold).
<em>
Emphasis (italic).
<b>, <i>
Presentational bold/italic.
<u>
Underlined text.
<small>
Smaller text.
<abbr title="…">
Abbreviation with tooltip.
<code>, <pre>
Code snippets / preformatted.

HTML: Lists

<ul> & <li>
Unordered list.
<ol> & <li>
Ordered list.
<dl>, <dt>, <dd>
Description list.

HTML: Links & Media

<a href="URL" target="_blank">
Hyperlink.
<img src="path" alt="desc" width height>
Image.
<video src controls></video>
Video player.
<audio src controls></audio>
Audio player.

HTML: Tables & Forms

<table>, <tr>, <th>, <td>
Table structure.
<form action="…" method="get/post">
Form tag.
<input type="text|email|password|checkbox…">
Inputs.
<textarea>, <select>, <option>, <button type="submit">
Other controls.

CSS: Syntax

selector {
  property: value;
}

CSS: Selectors

element
Tag selector.
.class
Class selector.
#id
ID selector.
element element
Descendant.
element > element
Child.
element + element
Adjacent sibling.
a:hover
Pseudo-class.
p::before
Pseudo-element.

CSS: Box Model

margin
Outside spacing.
border
Outline.
padding
Inner spacing.
width, height
Content box size.
box-sizing
content-box | border-box.

CSS: Typography

font-family
Font stack.
font-size
Text size.
font-weight
Thickness.
line-height
Line spacing.
text-align
Left/center/right/justify.
text-decoration
Underline/line-through.
color
Text color.

CSS: Background & Borders

background-color
Fill color.
background-image
URL or gradient.
background-size
cover | contain | auto.
border-radius
Rounded corners.
box-shadow
Shadow effect.

CSS: Layout

display
block | inline | inline-block | none.
position
static | relative | absolute | fixed | sticky.
top, right, bottom, left
Offsets.
overflow
visible | hidden | scroll | auto.

CSS: Flexbox

display: flex;
Container.
flex-direction
row | column.
justify-content
flex-start | center | space-between…
align-items
flex-start | center | stretch…
gap
Space between items.
flex: grow shrink basis
Item sizing.

CSS: Grid

display: grid;
Container.
grid-template-columns
e.g. 1fr 2fr.
grid-template-rows
e.g. auto 100px.
gap
Row/column gap.
grid-area
Named areas.

CSS: Colors & Units

Hex
#RRGGBB
rgb(), rgba()
Channel or with alpha.
hsl(), hsla()
Hue/sat/lightness.
px, em, rem
Absolute & relative sizes.
%, vw, vh
Percent & viewport units.
